# 五行珠宝每日佩戴推荐工具 - 产品设计文档

## 1. 产品概述

这是一个结合「传统天干五行」与「每日能量节律」的生活方式工具。旨在通过理性、审美与仪式感的方式，为用户提供每日珠宝佩戴建议。

## 2. 核心功能结构

### 2.1 用户输入层

- **出生信息录入**：
  - 公历出生日期 (YYYY-MM-DD)
  - 出生时辰 (00:00 - 23:59, 自动映射至十二时辰)
- **设置**（可选）：
  - 性别（虽主要看日主，但部分流派参考，此处暂忽略，保持极简）

### 2.2 核心计算引擎 (Core Engine)

- **日主计算模块**：
  - 输入：公历生日
  - 处理：调用 `lunar-javascript` 库转换公历 -> 农历 -> 四柱（八字）。
  - 输出：提取「日柱天干」（即“日主”）。
- **五行映射模块**：
  - 规则：甲乙-木, 丙丁-火, 戊己-土, 庚辛-金, 壬癸-水。
- **每日能量模块**：
  - 输入：当前系统日期。
  - 处理：计算今日的“日柱天干五行”与“日柱地支五行”。
  - 逻辑：判断今日五行与用户本命五行的生克关系（简化版）。
    - _设计策略_：不做吉凶判断，而是“能量互补”或“同频共振”。
    - 例如：日主为木，今日为水（水生木），则能量滋养；今日为金（金克木），则需“护盾”或“安抚”。

### 2.3 推荐策略系统

根据【用户本命五行】+【今日能量】生成建议：

- **规则矩阵**：
  - **缺补原则**：如果今日能量消耗日主（如木日遇金日），推荐“水”类珠宝（通关，金生水生木）或“木”类（比肩帮身）。
  - **增强原则**：如果今日能量生助日主，推荐“火”类（木生火，发挥才华）或“土”类（木克土，获取财富）。
  - _简化版初期策略_：主要基于**用户本命五行**进行“增强”与“平衡”推荐，辅以今日修正。

### 2.4 交互与UI层

- **风格定义**：新中式、极简、高级感。
- **关键页面**：
  - **首页/输入页**：极简表单，背景或许有水墨或矿石纹理。
  - **结果页 (签文卡片式)**：
    - 核心展示：今日五行关键词（如“木 - 生长”）。
    - 视觉重心：推荐的珠宝材质高清图/色块。
    - 文案：温和的生活方式建议。
    - 底部：相关商品入口（后期）。

## 3. 技术实现建议

### 3.1 技术栈

- **前端框架**：React / Next.js (易于组件化，SEO友好)。
- **样式库**：Tailwind CSS (快速构建高级感UI)。
- **核心库**：`lunar-javascript` (处理干支计算)。

### 3.2 伪代码逻辑

```javascript
import { Solar } from "lunar-javascript";

// 1. 获取日主
function getDayMaster(birthDateStr) {
  const solar = Solar.fromDate(new Date(birthDateStr));
  const lunar = solar.getLunar();
  const dayGan = lunar.getDayGan(); // 获取日干 (如 '甲')
  return dayGan;
}

// 2. 映射五行
const WU_XING_MAP = {
  甲: "木",
  乙: "木",
  丙: "火",
  丁: "火",
  戊: "土",
  己: "土",
  庚: "金",
  辛: "金",
  壬: "水",
  癸: "水",
};

// 3. 珠宝推荐数据库
const JEWELRY_DB = {
  木: {
    colors: ["绿色", "青色"],
    materials: ["翡翠", "绿松石", "孔雀石"],
    keywords: ["生长", "柔和", "自然"],
    desc: "今日适合佩戴充满生机的配饰，感受自然的呼吸。",
  },
  // ... 其他五行
};

// 4. 生成建议
function generateRecommendation(userProfile) {
  const dayMaster = getDayMaster(userProfile.birthDate);
  const element = WU_XING_MAP[dayMaster];
  const recommendation = JEWELRY_DB[element];

  return {
    userElement: element,
    dayMaster: dayMaster,
    ...recommendation,
  };
}
```

## 4. UI 设计细节建议

- **配色**：
  - 背景：米白 (#F5F5F0) 或 深灰 (#1A1A1A) 营造沉浸感。
  - 文字：衬线体 (Songti / Serif) 增加文化感。
  - 强调色：使用中国传统色（如：黛蓝、朱砂、藤黄）。
- **动效**：
  - 结果出现时使用缓慢的淡入 (Fade-in) 或 卷轴展开效果。
